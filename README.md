# Ranne R. Plaza — Personal Website

Personal portfolio website for **Ranne R. Plaza**, Information Systems graduate.

Built as a static site (HTML, CSS, vanilla JS) styled to match the RANNE PLAZA brand
(black, vivid red, light-gray geometric theme).

## Structure

- `index.html` — page content & layout
- `styles.css` — branding and responsive styles
- `script.js` — skill chips, mobile menu, scroll animations
- `assets/logo.png` — brand logo
- `assets/profile.jpg` — profile photo

## Run locally

```bash
python -m http.server 8000
```

Then open http://localhost:8000
